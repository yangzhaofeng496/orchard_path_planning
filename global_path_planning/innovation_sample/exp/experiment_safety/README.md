# 车辆安全裕度实验

固定30个圆形障碍物的果园地图及车辆几何尺寸，仅改变车辆周围附加安全裕度：0.00、0.15、0.30、0.45 m。每个等级生成5张地图，每张地图使用10个搜索种子，对比 RRT*、GoalBias 和 Hybrid，共600次规划。

- `benchmark_safety.py`：实验、断点续跑和绘图脚本
- `safety_results/`：运行后生成逐次CSV、汇总CSV和PNG/PDF/SVG图表
- 默认最大迭代数为1500，支持 `--max-iterations` 与 `--workers` 参数

该实验用于验证在更严格安全约束下，各规划方法的可行性和搜索效率。
