# 果园行间净间隙实验

本实验采用圆形树冠组成的必经通道，固定环境尺度和车辆参数，仅改变两侧树冠边界之间的净间隙：2.5、3.5、4.5、5.5、7.0 m。每个间隙生成5张地图，每张地图采用10个搜索种子，对比 RRT*、GoalBias 和 Hybrid，共750次规划。

- `benchmark_gap.py`：实验、断点续跑和绘图脚本
- `gap_results/`：运行后生成逐次CSV、汇总CSV和PNG/PDF/SVG图表
- 默认最大迭代数为1500，支持 `--max-iterations` 与 `--workers` 参数

该实验验证采样策略对果园局部窄通道和树冠间可通行性的适应性。
